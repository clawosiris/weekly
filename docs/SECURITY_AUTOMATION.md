# Security Automation

This scaffold includes:

- Dependabot updates for GitHub Actions.
- OpenSSF Scorecard analysis.
- Pull request and issue templates that warn against sharing secrets or private data.
- A CODEOWNERS file listing `@noraarrr-cloud` and `@llunved`.

Add CodeQL after source code is imported and the primary languages are known. Enabling CodeQL before importing code can create failing CI jobs for languages that are not present.
